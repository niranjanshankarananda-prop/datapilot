Good to Know

  • * Valid API key required — You need a free Groq, Together AI, or OpenRouter API key configured above before querying. Without it, queries will fail.
  • * Use column names in queries — For best results, reference actual column names from your data (shown as clickable pills in workspace).
  • * Simple queries work best — Filtering, top-N, averages, groupby, and counts are well-supported. Complex multi-step analysis may need rephrasing.
  • * In-memory storage — Uploaded data is stored in server memory. If the server restarts, you'll need to re-upload your file.
  • * Works with any CSV/Excel — The app adapts to any dataset structure. Column names and types are auto-detected.
